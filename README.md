# podcaster

## Production

To run the production build go to podcaster-web install the modules and run the serve script

```
cd podcaster-web/
npm install
npm run serve
```

or use npx

```
cd podcaster-web/
npx serve -s build
```

## Development

To run the development project go to podcaster-web install the modules and run start script

```
cd podcaster-web/
npm install
npm start
```
